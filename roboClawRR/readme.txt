Jammster is using the roboclaw 2x60A motor controller with USB
https://www.pololu.com/product/1499


folder description:
roboclaw.py contains an extended python interface for the roboclaw controller 

roboclawRR_server.py is a robot raconteur server for controlling the roboclaw

roboClaw_service.robodef is the corresponding service definition file for the RR server

roboclawClient.py contains an example for connecting to and using the RR server
